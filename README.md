# dragonflAI

AI-assisted bug bounty recon orchestration platform.

## Overview

dragonflAI automates reconnaissance for bug bounty programs by orchestrating security tools (subfinder, httpx, nmap, dnsx), storing results in PostgreSQL, and exposing a Streamlit UI for managing programs, targets, and runs.
